# 数据库管理

WebTerm 支持 MySQL 数据库的可视化管理，包括查询编辑、表浏览和内联编辑。

## 连接管理

### 添加数据库连接

1. 切换到数据库模块
2. 点击 **+** 新建连接
3. 填写主机、端口、用户名、密码、数据库名
4. 保存后点击连接

## 功能

### 数据库/表浏览

- 树形展示：数据库 → 表 → 列信息
- 列信息包含类型、索引、默认值

### 查询编辑器

- CodeMirror 6 + MySQL 方言高亮
- `Ctrl+Enter` 执行选中的 SQL 或全部 SQL
- 结果表格支持排序和分页（限制 1000 行）

### SQL 格式化

`Ctrl+Shift+F` 一键格式化 SQL，使用 sql-formatter 库。

### 内联编辑

- 双击结果表格中的单元格
- 编辑后回车提交 UPDATE

### 导出

- 查询结果导出 CSV

### 查询历史

- 本地存储最近 50 条查询
- 支持收藏常用查询

## 安全

- DDL 操作（ALTER/CREATE/DROP）需二次弹窗确认
- 服务端不记录完整查询日志（仅记录"发起查询"事件）
