# SSH 终端

WebTerm 基于 xterm.js 提供全功能终端模拟器，支持多标签页、无限分屏和广播模式。

## 连接管理

### 添加连接

1. 点击侧边栏 SSH 模块
2. 点击 **+** 新建连接
3. 填写主机地址、端口、用户名
4. 选择认证方式：密码 或 私钥
5. 保存后点击连接即可打开终端

### 连接分组

支持按分组管理连接，拖拽调整分组和排序。

## 终端功能

### 多标签页

- 点击连接打开新标签
- 标签支持拖拽排序
- 右键标签可关闭、关闭其他

### 无限分屏

- 右键标签 → 横向分屏 / 纵向分屏
- 每个面板独立标签栏
- 拖动分隔线调整面板大小
- 关闭面板后相邻面板自动扩展

### 广播模式

1. 选中一个标签作为"广播源"
2. 进入广播模式
3. 键盘输入同步发送到所有目标标签

适用于同时在多台服务器执行相同命令的场景。

### 主题定制

预设多套配色方案：
- Dracula
- Solarized Dark / Light
- One Dark
- Monokai

支持自定义前景色、背景色、ANSI 色板和光标样式。

### 关键字高亮

在设置中添加关键字和对应颜色，终端输出自动高亮。内置预设：

| 关键字 | 颜色 |
|---|---|
| ERROR | 红色 |
| WARN | 黄色 |
| INFO | 绿色 |
| DEBUG | 灰色 |

支持正则表达式匹配。

### Zmodem 文件传输

终端内使用 `sz` / `rz` 命令：
- `sz filename` — 发送文件（触发浏览器下载）
- `rz` — 接收文件（触发浏览器文件选择）

### 查找

`Ctrl+F` 打开终端内查找。

## 心跳与重连

- 服务端每 10 秒发送 WebSocket Ping 帧
- 客户端检测断连后指数退避重试（1s → 2s → 4s → 8s，最大 30s）
- 重连成功后复用原有 SSH 会话，若已断开则自动重新拨号
