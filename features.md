# 功能概览

WebTerm 将运维常用的工具整合到浏览器中，提供统一的操作界面。

---

## SSH 终端

基于 xterm.js 构建的终端模拟器，提供接近原生终端的使用体验。

### 多标签管理

- 支持创建多个 SSH 连接标签页，每个标签页独立运行
- 标签页标题自动跟随终端提示符（通过 OSC 7 协议）
- 拖拽调整标签页顺序

### 分屏布局

- 支持垂直和水平分屏，自由组合布局
- 分屏比例可拖拽调整
- 每个分屏独立管理自己的标签页

### 广播模式

- 三种广播范围：关闭、当前分屏、全部标签
- 一键向多个服务器同步发送命令
- 运维批量操作利器

### 终端特性

- 256 色 / TrueColor 支持
- Unicode 和 CJK 字符完美渲染
- Zmodem 文件传输协议
- OSC 7 当前目录同步（终端路径自动同步到 SFTP 面板）
- PROMPT_COMMAND 自动注入，无需手动配置

---

## SFTP 文件管理

双栏文件管理器风格，类 WinSCP 操作体验。

### 文件浏览

- 双栏布局，左右独立浏览不同目录
- 文件列表排序（名称、大小、修改时间）
- 目录树快速导航
- 显示文件图标和权限信息

### 文件操作

- 拖拽上传下载
- 批量文件上传
- 文件在线编辑（保存后自动上传）
- 新建文件/目录
- 重命名、复制、移动、删除
- 压缩解压

### 路径同步

- 在终端中 `cd` 切换目录时，SFTP 面板自动跟随
- 基于 OSC 7 协议，无需额外配置
- 手动切换 SFTP 目录不会影响终端

---

## 数据库管理

MySQL 数据库管理工具，提供便捷的查询和管理功能。

### 查询编辑器

- 语法高亮的 SQL 编辑器
- 多语句执行
- 查询结果分页展示
- 导出查询结果为 CSV

### 表管理

- 浏览数据库中所有表
- 查看表结构和索引信息
- 数据预览和内联编辑
- 快速生成 SELECT / INSERT 语句

### 连接管理

- 保存多个数据库连接配置
- 连接信息加密存储
- 快捷切换数据库

---

## 本地文件系统

- 浏览和管理服务器本地文件
- 与 SFTP 面板统一的操作体验
- 方便在本地和远程之间对比文件

---

## 主题系统

### 内置主题

- 多套精心设计的预设主题
- 亮色/暗色模式支持
- 终端主题独立于界面主题

### 自定义主题

- 支持自定义颜色方案
- 实时预览效果
- 配置持久化

---

## 更多特性

- **JWT 认证**：安全的用户认证体系
- **连接池管理**：SSH 连接引用计数，自动释放空闲连接
- **REST API**：完整的 HTTP API，方便集成
- **日志审计**：操作日志记录，满足合规需求
- **响应式设计**：适配桌面和移动设备
